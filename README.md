# E-Commerce Data Analysis

## Project Overview

This project demonstrates an end-to-end data analysis workflow using an e-commerce dataset. The analysis includes data preprocessing, exploratory data analysis (EDA), feature engineering, and predictive modeling using both Linear Regression and Logistic Regression.

The goal is to extract business insights from customer and sales data while applying machine learning techniques to solve regression and classification problems.

---

## Objectives

* Clean and preprocess e-commerce data
* Handle missing values and data quality issues
* Perform exploratory data analysis (EDA)
* Visualize trends and patterns in customer behavior
* Build and evaluate a Linear Regression model
* Build and evaluate a Logistic Regression model
* Interpret model results and business insights

---

## Dataset

The dataset contains e-commerce transaction and customer information such as:

* Customer demographics
* Purchase history
* Product information
* Transaction details
* Sales-related variables

*Note: Dataset is provided for educational and portfolio purposes.*

Used to predict continuous target variables.

* Root Mean Squared Error (RMSE)

### 2. Logistic Regression

Used for binary classification tasks.

**Evaluation Metrics**

* Accuracy
* Precision
* Recall
* F1 Score
